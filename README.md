## Project Overview

This Slack bot for AUK provides automated messaging capabilities with AI-powered document search and contextual responses.

### Core Functions
- **Message Processing**: Handles incoming Slack messages and routes them appropriately
- **User Management**: Stores conversation history
- **AI-Powered Responses**: Generates contextual replies using Vector DB search across uploaded documents
- **Session Management**: Maintains active conversation sessions

### Key Features
- Integration with Slack for team notifications
- Vector Database for PDF document storage and semantic search
- Conversation logging and archiving
- Error handling and retry mechanisms

### Technical Capabilities
- REST API endpoints for message handling
- Webhook support for real-time updates
- Vector DB integration for document embedding and retrieval
- Database persistence for user data
- PDF processing and indexing
- Environment-based configuration
